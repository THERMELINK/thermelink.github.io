Project.Text_Adventure()

Text adventure?

Een text adventure is een type spel waarbij je doormiddel van tekst jezelf navigeerd door het spel, hierbij gebruik je simpele commands om acties uit te voeren en uiteindelijk bij het einde te komen.

leerdoelen

De leerdoelen van dit project lagen meer code technisch en minder qua samenwerking, dit was een solo project waarbij we lieten zien dat we de basis kennis met de console onder de knie hadden.

Proces

tijdens dit project waren er ook wel uitdagingen, aangezien dit het eerste echte "game" project was waren er wel problemen zoals logica. Denk hierbij aan de volgorde wanneer iets moet gebeuren zoals het clearen van de console, om vervolgens de volgende frame te laten zien. ook was het maken van een game-plan totaal nieuw, daardoor waren er soms wel onduidelijkheden over features die ik niet compleet had uitgeschreven.

Eind Resultaat

Uiteindelijk was het zeker wel gelukt om een Text-adventure op te zetten!

Het eindresultaat is een text-adventure die zich afspeelt in een verlaten huis waar je opeens niet meer uit kan! het is de bedoeling dat je de sleutel vindt, en naar de uitgang komt zonder gepakt te worden door de enemy. maar je kan de enemy niet zien, tenzij je heel dichtbij bent!.

waar was ik voor verantwoordelijk?

tijdens dit project was ik verantwoordelijk voor het totale eindproduct, vandaar dat er ook een goede planning werd gemaakt om zo de taken te verdelen

daarnaast was ik ook verantwoordelijk voor het inleveren van alle documentatie en extra producten zoals cover art, basis ideeen en een bestandje die beschrijft hoe je het spel kan winnen
